# Anki Garden Cozy Hand-Painted SVG Pack

This starter pack is designed for direct use by Codex or a front-end build pipeline.

## Style
- Cozy hand-painted vector look
- Transparent backgrounds for props, plants, weather, and UI
- Strong visual growth separation between lifecycle stages

## Plant stage readability
- `seed` = tiny seed on soil
- `sprout` = simple stem with two cotyledons
- `young` = clear early structure, still sparse
- `mature` = dominant silhouette with full foliage
- `flowering` = mature silhouette plus blooms
- `rare` = premium evolution with glow, sparkles, or stronger ornamentation

## Integration notes
- Anchor plant placement from bottom center
- Preserve aspect ratio when scaling
- Use balanced weather assets by default and performance variants for low-power rendering
- Backgrounds are full-scene layers
- UI assets are decorative frames, badges, and meters

## Included directories
- `backgrounds/`
- `decorations/`
- `plants/`
- `ui/`
